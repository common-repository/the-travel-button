== Description ==

Instala nuestro plugin The Travel Button® para WordPress y monetiza tu contenido de viajes mientras ayudas a tus visitantes a contratar todos los servicios que puedan necesitar para organizar sus viajes, de forma simple e intuitiva.

Fácil de instalar, The Travel Button® es una herramienta gratuita y efectiva para generar ingresos, que te da acceso a Travelfinder, el metabuscador de servicios turísticos, accesible a través de botones personalizables.

Con Travelfinder nuestros afiliados pueden mostrar su logotipo, elegir los servicios que se quiera ofrecer y los proveedores con los que trabajar. Además, pueden asociarlo al destino que quieran.

Puedes ofrecer cualquier servicio que los viajeros puedan necesitar: hoteles, apartamentos, hostales, aviones, trenes, autobuses, coches de alquiler, restaurantes, actividades o experiencias.

Únete a la red de afiliación de We Travel Hub y usa el plugin para personalizar y gestionar los diferentes botones que quieras usar en tus publicaciones.

== Installation ==

Sube el plugin "The Travel Button®" a tu blog y actívalo.

Crea botones de viajes customizados y añadelos a tus publicaciones.

No olvides configurar tu código de website una vez que hayas validado tu website en [We Travel Hub](https://www.wetravelhub.com).

Si tienes alguna duda, por favor, visita nuestro [Help Center](https://help.wetravelhub.com).

== Main features ==

Beneficios para tus visitantes

* Ahórrales tiempo. Podrán consultar diferentes servicios turísticos sin abandonar tu web: transporte, alojamiento, restauración o actividades.
* Seguro. Contratarán los servicios con los mejores proveedores del sector turístico.

Beneficios para los creadores de contenido

* Respetuoso con el diseño de tu página web. Ajusta su estilo y colócalo dónde quieras.
* Rentable. Recibirás el 80% de las comisiones pagadas por los anunciantes con las reservas originadas en tu página web.

== How does it work? ==

1. Personaliza The Travel Button® con el diseño que más te guste. Puedes ver algunos ejemplos de cómo queda en una página web [aquí](https://www.wetravelhub.com/demo).
2. Asócialo a uno o varios destinos.
3. Configura Travelfinder en tu área privada.
4. Utiliza el editor de bloques de WordPress y añade el shortcode asociado a tu botón.

Cualquiera que inspire a la gente a viajar, o que tenga un negocio turístico que alguien pueda visitar, ahora tiene la oportunidad de generar una nueva fuente de ingresos de forma simple y transparente.
